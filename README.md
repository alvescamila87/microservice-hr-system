# Microservice HR-SYSTEM

## Sobre o projeto
Projeto do curso de microsserviço para aprender a desenvolver um sistema de microsserviços do zero, gerando as APIs de usuário e pagamento, com tratamento de erros.

### APIs implementadas
* User API
* Payroll API
* Eureka API

## Stacks
* Java 17
* Spring
* Eureka
* Banco H2
* Postman
